# Racing Project — Ducati Panigale V2 (Supersport)

Проект инженерной и тренерской поддержки гонщика.

- **Класс:** Supersport
- **Мотоцикл:** Ducati Panigale V2
- **Уровень:** топ-5 Чемпионата России
- **Треки сезона:** Москва Рейсинг Вэй (МРВ), Игора Драйв (Игора)

## Структура проекта

```
RacingProject/
├── engineering/          # Инженерная работа: сетапы, подвеска, данные
│   ├── SETUP.md         # Текущий актуальный сетап (быстрая ссылка)
│   ├── setups/          # Сетапы по трекам/условиям
│   ├── suspension.md    # Лог настроек подвески
│   └── data/            # Экспорт таблиц, телеметрия
└── coaching/             # Тренерская работа: техника, прогресс, трассы
    ├── PLAN.md          # Цели сезона, слабые места, план
    ├── tracks/          # Разбор трасс по секторам
    ├── sessions/        # Логи заездов
    └── riding-notes.md  # Техника: торможения, траектории, работа телом
```
