## Шпаргалка по эффекту регулировок

### Вилка (Öhlins NIX30)
| Симптом | Регулировка |
|---|---|
| Вилка проваливается на торможении, низ морды | + сжатие / + прелоад |
| Жёстко бьёт на тормозе, не загружает резину | − сжатие |
| Распускается после поворота, нестабильный вход | + отбой |
| Зависает, не успевает распрямиться в быстрой серии | − отбой |

### Амортизатор (TTX36)
| Симптом | Регулировка |
|---|---|
| Раскачка на выходе из поворота, squat | + сжатие / поднять начало клапана |
| Не выезжает из поворота, «задохликом» ощущается | − сжатие |
| Слип/потеря сцепления на ускорении | + отбой? проверить геометрию |
| Нервный rear на торможении | + отбой, проверить длина аморта |
